# Code of Conduct

This is a private project. All participants must be respectful, professional,
and security‑conscious.

## Expected Behavior
- Respect the owner’s decisions and project scope.
- Keep all project details confidential.
- Communicate clearly and professionally.

## Unacceptable Behavior
- Sharing code or sensitive details publicly
- Harassment, abusive language, or intimidation
- Bypassing security policies

## Enforcement
The owner has final authority on conduct decisions.
